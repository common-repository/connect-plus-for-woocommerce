Send orders to Olivery Connect Plus application to distribute them to delivery companies

== Description ==

Olivery Connect Pluse for WooCommerce is a powerful integration tool that streamlines your order 
fulfillment process by seamlessly connecting your WooCommerce store with the Olivery Connect application. 
This plugin allows you to send your orders to the Olivery Connect platform automatically, 
where they are intelligently distributed to a network of trusted delivery companies, ensuring 
timely and efficient delivery to your customers.

Requirements:

    A WooCommerce-powered WordPress site.
    
    An active account with Olivery Connect Plus.

== External Service Usage ==


This plugin uses the [Olivery Connect Plus](https://website.olivery.app/) API to integrate with their platform for order management. When using the plugin, the following actions will trigger data transmission to the external service:


- **Add Orders**: Sends order data to `https://api.connect-plus.app/integration/add_orders`.

For more information on the Olivery Connect Plus API, please refer to their [terms of use](https://website.olivery.app/terms-of-use/) and [privacy policy](https://website.olivery.app/privacy-policy/).


== Frequently Asked Questions ==

= Who can use this plugin ? =

This plugin is usefull for those who need to deliver their orders by using Olivery connect plus and allready have a Wordpress-Woocommerce store .

= How can i use this plugin ? =

After you install and activate the plugin you will see a new menu link named 'Olivery Connect', this is where you can fill your settings that you can get from Olivery Connect. beside that in your order edit admin page you will be able to see a button named 'send to Olivery Connect' , click on it and a copy of your order basic informatin will be sent to Olivery in order to proccess your order, you will also be able to see a new choices in orders bulk actions to do the same thing


= Do i need woocommerce? =

Yes,sure , this plugin have no meaning without woocommerce.
